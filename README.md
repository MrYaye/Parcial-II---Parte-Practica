## Descripción del Proyecto

Este proyecto es una **página web interactiva estilo Disney** desarrollada como parte del Parcial II - Parte Práctica. Se trata de un landing page responsivo que presenta contenido multimedia, promociones de Disney+ y ofertas de vacaciones en parques temáticos Disney.

## Características Principales

**Navbar Responsivo**
- Menú de navegación dinámico con scroll suave
- Barra de búsqueda interactiva
- Logo centrado en versiones móviles

**Hero Section**
- Banner de bienvenida con efecto parallax
- Botones de llamada a acción (CTA)

**Contenido Multimedia**
- Sección de características destacadas con tarjetas interactivas
- Catálogo de películas próximas en Disney+
- Ofertas de vacaciones en parques Disney
- Sección de promociones especiales

**Diseño Responsive**
- Adaptable a dispositivos móviles, tablets y desktop

**Estilo Visual Disney**
- Colores corporativos de Disney
- Footer con diseño moderno oscuro
- Efectos hover en botones y tarjetas
- Tipografía limpia y legible

**Interactividad**
- Navegación entre secciones con scroll suave
- Notificaciones visuales de acciones del usuario
- Búsqueda de contenido
- Efectos visuales en botones
